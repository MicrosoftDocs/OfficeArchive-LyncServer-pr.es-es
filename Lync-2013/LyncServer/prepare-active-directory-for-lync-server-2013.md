_**Última modificación del tema:** 2012-10-01_

Antes de implementar Lync Server 2013 en un estado de coexistencia con Office Communications Server 2007 R2, debe realizar otras tareas de Active Directory para configurar el esquema, el bosque y el dominio de Lync Server 2013. Las extensiones de esquema agregan las clases y atributos de Active Directory que necesita Lync Server. Para obtener más información, vea el tema sobre la [preparación de los servicios de dominio de Active Directory para Lync Server 2013](lync-server-2013-preparing-active-directory-domain-services.md).

**Preparar Active Directory para Lync Server 2013**

1.  En el servidor front-end de Lync Server 2013, ejecute la instalación de Lync Server 2013.

2.  Seleccione **preparar Active** Directory
    
    ![Asistente para la implementación de Lync Server 2013, Página principal](images/JJ205265.5f88ae18-9c3c-42ea-a91a-836ecf5d515f(OCS.15).jpg "Asistente para la implementación de Lync Server 2013, Página principal")

3.  Complete los pasos 1 a 5.
