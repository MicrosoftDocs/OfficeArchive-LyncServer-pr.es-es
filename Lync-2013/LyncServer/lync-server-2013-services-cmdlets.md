El rol de servidor y los cmdlets de servicios que se incluyen en Microsoft Lync Server 2013 le permiten devolver información acerca de los distintos componentes que se han instalado como parte de su implementación de Lync Server 2013. Además, estos cmdlets hacen que sea más fácil detener e iniciar los servicios de Lync Server.
