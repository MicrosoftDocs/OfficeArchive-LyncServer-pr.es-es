# <a name="deploying-devices-in-lync-server-2013"></a>Implementación de dispositivos en Lync Server 2013

_**Última modificación del tema:** 2013-09-10_

Lync Server 2013 incluye Lync Phone Edition, software que se ejecuta en dispositivos calificados y proporciona características de telefonía tradicionales y avanzadas, seguridad integrada, facilidad de administración y más. Lync Phone Edition funciona de la misma manera con Lync Server 2013 que con Lync Server 2010. Para obtener detalles sobre la implementación de dispositivos, consulte [implementación de Lync Phone Edition](https://go.microsoft.com/fwlink/p/?linkid=285880) en la biblioteca de TechNet de 2010 de Lync Server.
