# <a name="server-software-and-infrastructure-support-in-lync-server-2013"></a>Software de servidor y compatibilidad con la infraestructura en Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

_**Última modificación del tema:** 2012-06-29_

La compatibilidad de software para los componentes de servidor, incluidos el almacenamiento y los roles de servidor de Lync Server 2013, incluye sistemas operativos compatibles, software de base de datos, software de infraestructura y otro software necesario para admitir funcionalidades específicas. También incluye la virtualización de los componentes del servidor.
