_**Última modificación del tema:** 2012-10-09_

Los cmdlets de replicación proporcionan una manera de supervisar y administrar la replicación de Lync Server. Puede usar estos cmdlets para configurar las opciones de replicación; para supervisar el progreso de la replicación; y para forzar manualmente la replicación en un servidor.
